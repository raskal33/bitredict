🎲 How It Works​

Example: Football Pool​

// Pool: "Manchester City beats Real Madrid"

// Alice (Creator):
// - Stakes 500 STT AGAINST Man City winning
// - Sets 2.5x odds
const pool = await createPool({
  prediction: "Man City Wins",
  stake: 500,
  odds: 2.5,
  betAgainst: true
});

// Bob & Carol (Bettors):
// - Bet 200 STT total FOR Man City winning
await pool.placeBet({
  amount: 200,
  betFor: true
});

// Result:
// Man City wins: Bettors get 500 STT
// Real Madrid wins: Creator gets 700 STT
